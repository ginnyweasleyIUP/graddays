HadCRUT.4.6.0.0 (current version)

This is the current version of HadCRUT4. Previous versions of the dataset are available here.

  • The land ensemble has been updated to use station data for CRUTEM4 version CRUTEM.4.6.0.0.
  • HadSST3 version HadSST.3.1.1.0 is the source of SST data.
  • Data can be download here.
